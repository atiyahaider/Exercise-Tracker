Exercise tracker

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

To GET the users's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

For Example: /api/exercise/log/5c44d8c8d45c6035742edfe3/2018-02-01/2019-01-21/2

or:  /api/exercise/log/5c44d8c8d45c6035742edfe3/2018-02-01
